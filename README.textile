h1. Color Logger

h3. Info

p. Adds colors to your log files while maintaining default rails logging. The format has also been modified to:

[TYPE TIMESTAMP] [PROCESS_NAME.PID] MESSAGE

<pre>
	[INFO 09/24/09 12:26:59] [script/ftpserver.14736] Ready and waiting for connections
</pre>
